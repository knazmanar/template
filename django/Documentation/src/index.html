Introduction

Morvin is a fully featured premium admin dashboard template in Django with developer-friendly codes.

Morvin is an admin dashboard template that is a beautifully crafted, clean & minimal designed admin template with Dark, Light Layouts with RTL supported.

It will help your team moving faster and saving development costs and valuable time. If you’re a developer and looking for an admin dashboard that is fully responsive with Bootstrap and Django then you are at the right place to start your project using Morvin - Django Responsive Bootstrap ^5.0.0-beta2 Admin Dashboard.

Morvin contains lots of new design widgets with responsive on all screens. Also, there are 7 different types of Layouts we have added. It is very easy to change any layout in your existing running application by changing couple of lines code. We have written standard and developer-friendly code to increase performance.
